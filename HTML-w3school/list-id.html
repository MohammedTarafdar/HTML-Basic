HTML id Attribute

The HTML id attribute is used to specify a unique id for an HTML elements. You can not have more then one element with the same "id" in a HTML page.

Difference between HTML class and id

A class name can be used by multiple HTML elements, while an id name must only be used by one HTML element withen the page.
