Suggestion

Always use lowercase allributes

The HTML standard does not require lowercase attribute names. However, W3C recommends lowercase attributes in HTML.

Always quote attribute values

The HTML standard does not require quotes around attribute values. However, W3C recommends quotes in HTML.
